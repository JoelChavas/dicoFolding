dicoFolding.postprocessing.eval_tools.compute_loss(dico_set_loaders, model, loss_type, root_dir)¶

Returns list of loss values for each dataset_loader batch dataset_loader: dataset on which compute loss model: trained model used to compute loss loss_type: loss function to use to compute loss (L1, L2, SSIM) path_mode: if True (possible to track loss to samples) show: if True, display output images in Anatomist

Print outliers of a given model for each dataset of dico_set_loaders IN:

populations to compare and values, associated dataloader model: model trained
